# 开发规范

## vuex 模块（module）的命名用下划线

good

```js
const play_log = {
  state: {},
  mutations: {},
  actions: {},
  getters: {}
};

const store = new Vuex.Store({
  modules: {
    play_log
  }
});
```

bad

```js
const playLog = {
  state: {},
  mutations: {},
  actions: {},
  getters: {}
};
```

bad

```js
const PlayLog = {
  state: {},
  mutations: {},
  actions: {},
  getters: {}
};
```

## 模块（module）的文件名和模块名保持一致

## ipc method 名字使用 `s_xxx`

goods

```js
ipcEvent.sender.send("home", {
  method: "s_navigate",
  data: "xxx"
});
```
